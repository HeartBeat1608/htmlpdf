# Code of Conduct

## Our Standard

We want this project to be welcoming, respectful, and collaborative.

Examples of positive behavior:

- being respectful in disagreement
- giving constructive feedback
- assuming good intent while staying clear and honest
- helping other contributors learn and succeed

Examples of unacceptable behavior:

- harassment or abusive language
- personal attacks
- sustained disruptive behavior
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, issues, pull requests, or other contributions that violate this code of conduct.

## Reporting

Please report conduct concerns privately to the maintainers through the repository security/contact channels once they are configured.
